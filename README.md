# 📊 Hunter Sales KPI Dashboard

Dashboard interactivo de **KPIs de ventas SaaS** diseñado para demostrar cómo los equipos de ventas tipo **Hunter** analizan el rendimiento comercial mediante indicadores clave.

El proyecto permite **introducir datos manualmente** y visualizar cómo cambian los KPIs en **tiempo real** a través de gráficos dinámicos.

# 📈 KPIs incluidos en el dashboard

El panel muestra **8 indicadores clave utilizados en empresas SaaS** para analizar el rendimiento de ventas:

### 1️⃣ MRR (Monthly Recurring Revenue)

Ingresos recurrentes mensuales generados por nuevos clientes.

### 2️⃣ CAC (Customer Acquisition Cost)

Costo promedio de adquirir un cliente nuevo considerando gastos de marketing y ventas.

### 3️⃣ LTV / CAC Ratio

Relación entre el valor del cliente a lo largo del tiempo y el costo de adquisición.

### 4️⃣ Conversion Rate

Porcentaje de leads que se convierten en clientes.

### 5️⃣ Win Rate

Porcentaje de oportunidades de venta ganadas.

### 6️⃣ Pipeline Value

Valor total de las oportunidades activas en el proceso de ventas.

### 7️⃣ Churn Rate

Porcentaje de clientes que abandonan el servicio.

### 8️⃣ Sales Velocity

Velocidad con la que el pipeline genera ingresos.

# 🧠 Objetivo del proyecto

El objetivo de este proyecto es demostrar cómo se pueden visualizar y analizar **métricas clave de ventas en empresas SaaS** mediante un dashboard interactivo.

Este tipo de dashboards es utilizado por:

* equipos de **Sales Strategy**
* **Revenue Operations (RevOps)**
* **Sales Leaders**
* empresas **SaaS B2B**
