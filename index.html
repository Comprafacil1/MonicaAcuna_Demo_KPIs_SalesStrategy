⚡ ALERTAS CRÍTICAS

📊 Leads Generados
Para qué sirve: Volumen de oportunidades entrantes calificables.
Fuente datos: Formularios web + LinkedIn + Referidos.
Objetivo análisis: >100/mes mantiene pipeline saludable.
🎯 Tasa Conversión Leads→Oportunidad
Para qué sirve: Eficiencia proceso calificación inicial.
Fuente datos: Movimiento Leads→Oportunidades en CRM.
Objetivo análisis: >30% indica proceso de ventas elite.
💰 Pipeline Value Total
Para qué sirve: Valor monetario total en proceso venta.
Fuente datos: Suma valores estimados de oportunidades activas.
Objetivo análisis: >$50K constante = ingresos próximos 90 días.
🔮 Forecast Ventas
Para qué sirve: Proyección ingresos próximos 30-90 días.
Fuente datos: Pipeline × Win Rate × Factor Temporización.
Objetivo análisis: Prioriza acciones semanales críticas.
🏆 Win Rate
Para qué sirve: % cierres exitosos vs oportunidades totales.
Fuente datos: Oportunidades Ganadas / Total Oportunidades.
Objetivo análisis: >25% = performance vendedor profesional.
⏱️ Ciclo de Venta
Para qué sirve: Días promedio lead hasta cierre.
Fuente datos: Diferencia fechas creación-cierre oportunidades.
Objetivo análisis: <45 días = proceso eficiente optimizado.
🌍 Revenue Regional
Para qué sirve: Identifica zonas más rentables.
Fuente datos: Revenue agrupado por región geográfica.
Objetivo análisis: Aplicar regla 80/20 para foco estratégico.
🚨 Alertas Automáticas
Para qué sirve: Previene pérdida oportunidades por inacción.
Fuente datos: Reglas business vs métricas actuales.
Objetivo análisis: Respuesta <2h salva deals en riesgo.
